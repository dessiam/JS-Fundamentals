# JS-Fundamentals

This repository contains a collection of beginner-friendly JavaScript scripts designed to help understand core JavaScript concepts such as variables, loops, conditionals, and command-line arguments.

Each script is executed using Node.js and demonstrates a specific concept in a simple, practical way.
